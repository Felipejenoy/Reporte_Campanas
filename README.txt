# Reportes de Campañas Automatizados con Dashboard Web

Proyecto que combina **Python** para la generación de datos simulados de campañas y **JavaScript/HTML/CSS** para la visualización en un **dashboard web interactivo**.  

Se generan métricas de marketing en formato **JSON** y luego se muestran en gráficas dinámicas con **Chart.js**, accesibles desde la web gracias a **GitHub Pages**.

---

## Tecnologías utilizadas
- **Python** + **Pandas** → simulación de datos y cálculo de métricas.  
- **JSON** → almacenamiento e intercambio de datos.  
- **JavaScript** → carga de datos en el navegador.  
- **Chart.js** → visualización con gráficas interactivas.  
- **HTML5 / CSS3** → interfaz web.  
- **GitHub Pages** → despliegue en la web.  

---

##  Flujo del proyecto
1. **Generación de datos**  
   - Se simulan campañas de marketing.  
   - Se calculan métricas como **impresiones, clics, conversiones, CTR y tasa de conversión**.  
   - Los resultados se guardan en un archivo JSON.  

2. **Visualización en la web**  
   - Se cargan los datos en un **dashboard interactivo**.  
   - Se muestran gráficas dinámicas y fáciles de interpretar.  

3. **Despliegue en línea**  
   - El dashboard está publicado en **GitHub Pages**, accesible desde cualquier navegador.
